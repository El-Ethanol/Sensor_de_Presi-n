# Sensor de Presión.
#### Ethan Campos Méndez FC-UNAM 2021.

Programa (escrito en Python) para controlar y obtener datos del sensor de presión MKS PDR-900. 

Este está optimizado para el sensor ubicado en el ICN-UNAM. 

## Requerimientos: 

- PyQt5

- Python 3.x 

- Pandas

- pyqtgraph
